指令 Directive

在AngularJs视图模板中提供对HTML进行扩展，每一个指令都为HTML提供了一种新的能力，并且让模板更加语义化。在AngularJs中已经内置了一组指令，它们可以提供：

  • DOM节点的控制（ng-show, ng-hide, ng-if，ng-include, ...)
  • 渲染、循环数据（ng-bind, ng-repeat, ...）
  • 执行数据绑定（ng-model, ng-options, ...）
  • 关联控制器（ng-controller）
  • 其它...

控制器 Controller

管理一块视图区域，为视图模板提供访问服务层的一个接口。

依赖注入 DI

依赖注入是一种软件设计模式，用来处理代码的依赖关系，而不用关心被依赖对象的来源以及如何创建一个被依赖的服务。

推断 .controller(‘mainCtrl’, function($scope) { … });

内嵌 controller(‘mainCtrl’, [‘$scope’, function($scope){ … }]);

标记 MainCtrl.$inject = [‘$scope’]; function MainCtrl($scope) { … }

gulp插件 /* @ngInejct */function MainCtrl($scope) { … }
